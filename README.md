# ClientFlow AI 💼🤖

ClientFlow AI is a smart AI-powered assistant designed to help salons automate customer queries and appointment bookings.

---

## 🚀 Features

- 💬 AI Chatbot for customer interaction  
- 📅 Basic booking assistance  
- 🧠 Context-aware conversations (memory enabled)  
- ⚡ Fast and clean UI  
- 🌐 Ready for web deployment  

---

## 🧠 Use Case

This solution helps salons:
- Answer common customer questions instantly  
- Reduce manual workload  
- Handle booking requests efficiently  
- Improve customer experience  

---

## 🛠️ Tech Stack

- Python (Flask)
- Ollama (Local LLM - LLaMA 3)
- HTML, CSS, JavaScript (Frontend)
- REST API integration

---

## ⚙️ Setup Instructions

### 1. Clone the repository

git clone https://github.com/Talha-web-ai/ClientFlow.ai.git

cd ClientFlow.ai

### 2. Create virtual environment
python3 -m venv venv
source venv/bin/activate

### 3. Install dependencies
python3 -m pip install -r requirements.txt

### 4. Run Ollama (in another terminal)
ollama run llama3

### 5. Start the app
python3 app.py

### 6. Open in browser
http://127.0.0.1:5000
